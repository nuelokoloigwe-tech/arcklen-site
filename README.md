# Arcklen Group Limited Website

Official website project for Arcklen Group Limited.

## Overview

This project powers the Arcklen Group consulting website.

Arcklen focuses on:

- Business Analysis Consulting
- Process Improvement
- Change & Transformation Support
- BA Career Coaching

## Tech Stack

React  
TypeScript  
Vite  
Framer Motion  

## Development

Install dependencies:

npm install

Run the development server:

npm run dev

## Build

Create a production build:

npm run build

## Deployment

This project is deployed on Vercel.

Each push to the main branch triggers an automatic deployment.
